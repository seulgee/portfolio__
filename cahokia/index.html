Cahokia Mounds Informational Pamphlet
2017

This is a project that incorporates paper engineering as a crucial part of its design. I designed a mailable pamphlet that informs the user about the Cahokia Mounds State Historic Site in Illinois, focusing on portraying the vast scale of the site with statistical numbers and photographs. As the user unfolds the pamphlet, the information reveals itself bit by bit.

Fonts In Use

Adobe Illustrator, Adobe Photoshop, and Adobe InDesign were used to create this project.
