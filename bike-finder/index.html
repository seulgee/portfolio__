Quick Cycle
2018

This is a project with a focus on how to optimize the user's experience when shopping for the perfect bike. Assuming that the user has minimal knowledge on bicycles, Quick Cycle would guide the user through the decision making process by organizing the bike types by the bigger and more general categories on the Home Page, where a brief description of the bike type would be presented to the user on hover. This way, the user has an easier approach to a foreign subject. However, for the case of an informed user, there is the option of diving straight into it through the search bar. The Live Chat button lives as a fixed element for the user to immediately address their questions or concerns on any bike-related matter.

❮ ❯

In building this project, I went through the process of:

  • observing bike-consumer behavior
  • subject research
  • defining multiple personas and creating user stories for them
  • feature organization
  • wireframing
  • digital mockups
  • clickable prototyping using hotspots

Adobe InDesign, HTML, and CSS were used to create this project. Branding and customer reviews were not the focus of this project.
