Cigarettes & Cheetos
2017

This project is a concept development for an iPhone game. I created five distinct characters: a clerk, two robbers, and an honest customer who morphs into a shoplifter, which is indicated by a visual shift in the appearance of the character. In the game, the user experiences these characters taking action in two different settings, a convenience store and a gas station.

The Character Set (from left to right): Fiesty Granny (Robber 1), Sad College Boi (Robber 2), Evil Shoplifter, Honest Customer, Tired Girl (The Clerk)

Adobe Illustrator was used to create this project.
